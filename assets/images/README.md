# Images Directory

Place your image files in this directory with the following names:

## Required Images:

1. **logo.jpg** - The logo image for the top left header (circular, will be displayed in a round frame)
   - Recommended size: 200x200px or larger (square format)
   - Will be automatically cropped to fit the circular container

2. **hero-background.jpg** - Background image for the hero section (main banner)
   - Recommended size: 1920x1080px or larger
   - Used as a full-width background image

3. **about-image.jpg** - Image for the "About Us" section
   - Recommended size: 800x800px or larger (square format)
   - Will be displayed with an arch shape at the top

4. **pineapple-tarts.jpg** - Image for Pineapple Tarts menu item
   - Recommended size: 500x400px or larger
   - Displayed in menu card

5. **onde-onde.jpg** - Image for Onde-Onde menu item
   - Recommended size: 500x400px or larger
   - Displayed in menu card

6. **kuih-lapis.jpg** - Image for Kuih Lapis menu item
   - Recommended size: 500x400px or larger
   - Displayed in menu card

7. **curry-puffs.jpg** - Image for Curry Puffs menu item
   - Recommended size: 500x400px or larger
   - Displayed in menu card

## Notes:
- All images should be in JPG format
- Images will be automatically optimized by the browser
- Make sure images are properly compressed for web use to ensure fast loading times
